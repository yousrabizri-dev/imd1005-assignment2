## Assignment 3 Features Implemented
For Assignment 3, I completed 5 of the available JavaScript requests:

1. **Back-to-top button**  
   A floating button appears after scrolling and smoothly returns the user to the top of the page.

2. **Responsive hamburger menu**  
   On smaller screens, the navigation switches to a hamburger menu that can be opened and closed with JavaScript.

3. **Form validation**  
   The contact form validates multiple fields before submission.  
   Error messages appear next to invalid fields, disappear when corrected, and a success message appears when the form is valid.

4. **Dynamic content rendering from data**  
   The portfolio section is generated from a JavaScript array of objects instead of being hardcoded in the HTML.

5. **Live filtering by category**  
   Users can filter portfolio projects by category, and the active filter is visually highlighted.
   The Users can aslo search the projects by typing a keyword.
